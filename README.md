# Math Validator

Uniwersalny walidator równań matematycznych.

Waliduje:
- składnię
- algebrę
- logikę
- topologię (defekty, nieciągłości, osobliwości)
- numerykę (rozwiązania)

## Uruchomienie


## Przykład
Enter equation: x^2 - 4
{
"syntax": "OK",
"algebra": "OK",
"logic": "OK",
"topology": "OK",
"numeric": "Solutions: [-2, 2]"
}
